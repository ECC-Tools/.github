## The Ecosystem

### [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) — 99K+ stars

The most comprehensive AI coding plugin. One install gives you 30 agents, 138 skills, 60 commands, and production-ready hooks across Claude Code, Cursor, Codex, OpenCode, Gemini CLI, and Antigravity.

```bash
curl -fsSL https://ecc.tools/install | bash
```

### [ECC Tools](https://github.com/apps/ecc-tools) — GitHub App

Hosted GitHub App that analyzes repo history and agent configurations, opens PRs with repo-specific skills and safety checks, and runs hosted PR audits.

- Repo analysis, harness audits, setup guidance
- Team billing: per-seat, installation-pooled
- AgentShield-backed security scanning on PRs

### [AgentShield](https://github.com/affaan-m/agentshield) — Security Scanner

Scans AI agent configs for hardcoded secrets, overly permissive permissions, hook injection risks, and MCP vulnerabilities. 102 static rules, 1,282 tests, 98% coverage.

```bash
npx ecc-agentshield scan
```
