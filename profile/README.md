# ECC Tools

ECC Tools is the hosted GitHub App for bringing ECC into real repositories.

It analyzes repo history and agent-related configuration, opens pull requests with repo-specific skills, rules, hooks, commands, manifests, and safety checks, and runs hosted PR audits and selective rollout planning for AI coding agents.
