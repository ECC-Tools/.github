# ECC Tools

**The open-agent-harness ecosystem behind 182K+ GitHub stars.**

Start with the open-source repo. Add the GitHub App when you go private. Add AgentShield when security and rollout become first-order problems.

## 🚀 Install in 30 seconds

```bash
curl -fsSL https://ecc.tools/install | bash
```

Works on Claude Code, Cursor, Codex, OpenCode, Gemini CLI, Antigravity, and GitHub Copilot.

## 🧰 The Ecosystem

### [Everything Claude Code](https://github.com/affaan-m/everything-claude-code)
The agent-harness performance optimization system. **182K+ ⭐ · 28K forks.**

- 30+ agents · 138 skills · 60 commands
- Production-grade hooks, security, evals
- Cross-harness: Claude Code, Cursor, Codex, OpenCode, more

### [ECC Tools GitHub App](https://github.com/apps/ecc-tools)
Hosted GitHub App for private repos, recurring PR audits, AgentShield-backed security on every push, and team-pooled billing.

- **Free** — Public repo analysis, 10 runs/mo, 200 commits/run
- **Pro** ($19/seat/mo) — Private repos, 50 analyses/seat/mo, 1K commits/run, priority support
- **Enterprise** — Unlimited, SSO, custom rules, dedicated onboarding

[**See pricing →**](https://ecc.tools/pricing) · [**Install the App →**](https://github.com/apps/ecc-tools)

### [AgentShield](https://github.com/affaan-m/agentshield)
Security scanner for AI agent configs. **638 ⭐ · 1,609 tests · 98% coverage.**

```bash
npx ecc-agentshield scan
```

Finds hardcoded secrets, overly permissive permissions, hook-injection risks, MCP vulnerabilities. Featured at Cerebral Valley × Anthropic.
